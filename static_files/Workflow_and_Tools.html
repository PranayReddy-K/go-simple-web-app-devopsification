Tools Used

  • CI/CD Tools:

    GitHub Actions: Automates the build, test, and deploy processes. It integrates seamlessly with GitHub repositories and allows for complex workflows.
    Jenkins: An open-source automation server used for continuous integration and continuous delivery. Highly customizable with plugins.

  • Container Registry:

    Docker Hub: A popular public Docker registry where Docker images can be stored and shared. It provides a central repository for images.
    Amazon ECR: A managed Docker container registry service that allows you to store, manage, and deploy Docker container images securely and at scale.

  • Continuous Deployment (CD) Methods:

    Argo CD: A declarative, GitOps continuous delivery tool for Kubernetes that automates the deployment process.
    Traditional Methods: Shell scripts or Ansible playbooks can be used for automating deployment tasks outside of the GitOps approach.

  • Cluster Management:

    Amazon EKS: A managed Kubernetes service that simplifies the deployment, management, and scaling of containerized applications.
    Google Kubernetes Engine (GKE): A managed Kubernetes service from Google Cloud that provides a powerful environment for containerized applications.
